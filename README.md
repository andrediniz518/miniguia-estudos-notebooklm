# miniguia-estudos-notebooklm

## Lógica de programação com Python

Esse projeto tem como objetivo criar um caderno temático utilizando NotebookLM,
sobre Lógica de Programação com Python. A proposta é utilizar a IA como apoio aos
estudos, reunindo materias de referencia, testando diferentes prompts e organizar
um guia de estudo.

**Obejtivos**
* Revisar lógica de programação
* Criar um material de futuras revisões
* Utilizar IA como apoio

- Explique o que é lógica de programação para um iniciante que nunca programou antes, utilizando exemplos
do cotidiano em python para melhor entendimento. Depois me passa um exercicio simples para ver se eu entendi.

## Troubleshooting

### Dificuldade encontrada

O primeiro prompt gerou uma resposta correta, porém muito ampla e sem exemplos práticos em Python, dificultando a compreensão para um iniciante.

### Como resolvi

Refinei o prompt especificando que a explicação deveria utilizar exemplos do cotidiano, exemplos em Python e incluir um exercício de fixação.

### Resultado

Após o refinamento, a IA produziu uma resposta mais organizada, didática e adequada ao meu nível de conhecimento.

### Aprendizado

Aprendi que prompts específicos produzem respostas mais completas e úteis para o estudo.

## Variação do Prompt 2

Crie 5 exercícios de lógica de programação em Python com exemplos, em ordem crescente de dificuldade,
 sem fornecer as respostas.

**Resposta da IA:**

A IA criou cinco exercícios detalhado com exemplos de Python, abordando variáveis, condicionais, laços de repetição,

## Troubleshooting

**Dificuldade encontrada**

O prompt inicial gerou exercícios corretos, porém com pouca organização e sem uma progressão clara de dificuldade, dificultando a prática gradual dos conceitos.

**Como resolvi**

Refinei o prompt solicitando que os exercícios fossem organizados em ordem crescente de dificuldade e abordassem variáveis, condicionais, laços de repetição e funções, sem fornecer as respostas.

**Resultado**

Após o refinamento, a IA gerou cinco exercícios mais estruturados, com exemplos em Python e níveis de dificuldade progressivos, tornando a prática mais eficiente.

**Aprendizado**

Aprendi que definir o nível de dificuldade, os conteúdos desejados e o formato da resposta ajuda a IA a produzir exercícios mais organizados e adequados ao processo de aprendizagem.

## 📖 Glossário

| Termo | Definição |
|--------|-----------|
| **Lógica de Programação** | Conjunto de regras e instruções para resolver um problema de algoritmo. |
| **Variável** | Espaço na memória para armazenar informações. |
| **int** | Representa números inteiros |
| **float** | Representa números com casas decimais |
| **str** | Representa texto |
| **bool** | Valor lógico que possui True ou False |
| **if** | Executa um bloco de código se a condição for verdadeira |
| **elif** | Testa uma nova condição se a anterior for False |
| **else** | Quando nenhuma estrutura é atendida é executada |
| **for** | Estrutura de repetição para percorrer um elemento ou repetir uma ação |
| **while** | Executa um bloco de código até ela se tornar False. |
| **Função** | É um bloco de código reutilizavel chamada através do nome da função. |
| **Indentação** | Utilizado para definir em qual bloco de código as instruções vão estar |
